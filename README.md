# Asger Storgaard Høffner – Portfolio

Personlig portfolio- og kursusside for **AI-driven Applications (AIDA)**-forløbet på CPH Business Academy.

Siden viser udvalgte projekter, en ugentlig logbog over AIDA-kurset samt information om mig som datamatikerstuderende.

Bygget med [Hugo](https://gohugo.io/) og [Hextra](https://github.com/imfing/hextra)-temaet. Deployet via Netlify.

## Indhold

- **Forside** – Intro, tech-stack og direkte links
- **AIDA-Portfolio** – Ugentlig dokumentation af kursets indhold og implementeringer
- **Projekter** – Udvalgte projekter fra studiet
- **Om mig** – Baggrund, kompetencer og kontakt

## Lokal udvikling

Kræver [Hugo](https://gohugo.io/getting-started/installing/) og [Go](https://golang.org/doc/install).

```shell
git clone https://github.com/AsgerSH/Portfolio.git
cd Portfolio
hugo server -D
```

Siden kører på `http://localhost:1313/` (eller næste ledige port).
